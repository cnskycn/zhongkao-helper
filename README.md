# 莞深初升高 · 志愿填报助手

一个聚合深圳和东莞高中**录取分数线、高考成绩、中考政策**的查询工具。

👉 **在线访问**：https://cnskycn.github.io/zhongkao-helper/

## 功能

| 模块 | 内容 |
|------|------|
| 📊 分数线 | 深圳 80+ 所公办高中 3 年分数线对比，东莞 100+ 条（公办民办/本市户籍/随迁子女/公费/自费） |
| 🏆 高考成绩 | 深圳 + 东莞 50+ 所高中 2025 年高考数据（特控率、本科率、985/211 率） |
| 📋 中考政策 | 深圳 + 东莞 2024-2026 年中考政策汇编 |

## 数据来源

- ima 知识库「东莞深圳高中数据」
- 深圳市招生考试办公室
- 东莞市教育局

## 技术栈

单页 HTML 应用，无依赖，零部署成本。数据由知识库 + AI 自动读取生成。

## 本地使用

```bash
# 直接打开
open index.html

# 或启动本地服务器
python -m http.server 8000
# 访问 http://localhost:8000
```
